 1. What is Semantic HTML? 
A: It is HTML tags that supply some meaning to their content
    2. What is HTML used for? 
A:It is a markup language used to display data
    3. What is an attribute and where do we put it? 
A: An attribute is a special flag inside a tag (such as a link or image source) that contains data we don't want to display on a page.
    4. What is the h1 tag used for? How many times should I use it on a page?
A:It marks the most important header. It should only be used once.
    5. Name two tags that have required attributes
The anchor tag <a> and the image <img> tag
    6. What do we put in the head of our HTML document?

    10. What is a class?
